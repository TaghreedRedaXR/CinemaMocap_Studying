Cinema Mo Cap now supports Kinect v1.0 for Xbox360 and PC, as well as Kinect 2.0 for Xbox One and PC (With required USB power cable and adaptor)
===========================

Kinect v1.0 Instructions:
----
You MUST install the Micrsoft Kinect SDK before you can run this application!

1. Download the Microsoft Kinect® v1.8 SDK at the following location:

	http://www.microsoft.com/en-ca/download/confirmation.aspx?id=40278

2.	Make sure your Kinect® is UNPLUGGED FROM YOUR COMPUTER.  This is very important.  If this step is not done, the Kinect® drivers will not properly install.

3.	Run the executable file.  The SDK process is a one button install, simply accept the license terms and click the install button.

4.	Click Close when prompted.  Note: you do not need to install the Developer Toolkit.

5.	For install instructions from Microsoft, see the following link:
	http://msdn.microsoft.com/en-us/library/hh855354.aspx#Installation
===========================

Kinect v2.0 Instructions:
----
1. Requirements:

	- YOU MUST have a Windows 8 or 8.1 machine to support the Kinect 2.0 hardware device. 
	- USB 3.0 port available on your computer.
	- 64-bit processor
	- 4 GB of memory or higher (8 GB recommended)
	- 3.1 Ghz or faster multi-core processor
	- DX11 capable graphics adaptor

2. Download the Microsoft Kinect® v2.x SDK at the following location:

	http://www.microsoft.com/en-us/download/details.aspx?id=44561

3.	Make sure your Kinect® is UNPLUGGED FROM YOUR COMPUTER.  This is very important.  If this step is not done, the Kinect® drivers may not properly install.

4.	Run the executable file.  The SDK process is a one button install, simply accept the license terms and click the install button.

5.	Click Close when prompted.  Note: you do not need to install the Developer Toolkit.

6.	For install instructions from Microsoft, see the following link:
	http://msdn.microsoft.com/en-us/library/hh855354.aspx#Installation
===========================
